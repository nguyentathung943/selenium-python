*To specify all the test case
1. Make sure python version is newer than 3.8

2. install python library: pip install selenium

3. Install 3 browsers Firefox, Edge, Chrome and download the web driver corresponding
to the browser version:
Here i use:
Chrome: Version 92.0.4515.107
-> Web driver: https://chromedriver.storage.googleapis.com/index.html?path=92.0.4515.107/
Egde: Version 92.0.902.62
-> Web driver: https://msedgedriver.azureedge.net/92.0.902.62/edgedriver_win64.zip
Firefox: Version 92.0.2
-> Web driver: https://github.com/mozilla/geckodriver/releases/download/v0.29.1/geckodriver-v0.29.1-win64.zip

4. Put all of these drivers in the Driver folder
5. Compile by VSCode and run 18127104.py or type python 18127104.py to the console
